Programación Orientada a Objetos: Abstracción

# ¿Que es la Abstracción?


Es el proceso de seleccionar las características y funciones más relevantes de una entidad real para modelarla como una clase. Por ejemplo, si queremos representar un automóvil, podemos abstraer sus atributos como marca, modelo, color, velocidad, etc., y sus métodos como arrancar, frenar, acelerar, etc. La abstracción nos permite simplificar la complejidad del problema y enfocarnos en lo esencial. 

    
        class Vehiculo {
            constructor(marca, modelo) {
              this.marca = marca;
              this.modelo = modelo;
              this.encendido = false;
            }
          
            encender() {
              if (!this.encendido) {
                this.encendido = true;
                console.log(`El ${this.marca} ${this.modelo} ha sido encendido.`);
              } else {
                console.log(`El ${this.marca} ${this.modelo} ya está encendido.`);
              }
            }
          
            apagar() {
              if (this.encendido) {
                this.encendido = false;
                console.log(`El ${this.marca} ${this.modelo} ha sido apagado.`);
              } else {
                console.log(`El ${this.marca} ${this.modelo} ya está apagado.`);
              }
            }
          }
          
          // Crear objetos de la clase Vehiculo
          const miAuto = new Vehiculo("Toyota", "Corolla");
          const miMoto = new Vehiculo("Honda", "CBR");
          
          // Interactuar con los objetos
          miAuto.encender();
          miMoto.encender();
          
          miAuto.apagar();
          miMoto.apagar();

## Su Sintaxis


La abstracción en la programación orientada a objetos (POO) se implementa a través de la creación de clases y objetos que representan conceptos abstractos o entidades del mundo real. Aquí tienes una descripción de la sintaxis básica para implementar abstracción en POO:


Definición de clases


Una clase es una plantilla que define la estructura y el comportamiento de un objeto. Puede incluir atributos (propiedades) y métodos (funciones) relacionados. Aquí está la sintaxis básica para definir una clase:


Python

    
    class NombreDeLaClase:
        def __init__(self, parametros):
            # Constructor para inicializar atributos
            self.atributo = valor

        def metodo(self, parametros):
            # Implementación del método
            pass

Creación de objetos


Un objeto es una instancia de una clase. Se crea utilizando el constructor de la clase. Aquí está la sintaxis para crear un objeto:


Python

    
        objeto = NombreDeLaClase(argumentos)

Atributos y Métodos:

Los atributos son variables que almacenan datos relacionados con el objeto, mientras que los métodos son funciones que definen el comportamiento del objeto. Pueden ser públicos (accesibles desde fuera de la clase), protegidos (accesibles desde la clase y sus subclases) o privados (accesibles solo desde la misma clase).


Python

    
        class Persona:
        def __init__(self, nombre, edad):
            self.nombre = nombre  # Atributo público
            self.__edad = edad    # Atributo privado
    
        def obtener_edad(self):
            return self.__edad
    
        def __metodo_privado(self):
            pass

Herencia y Polimorfismo:

La herencia permite crear nuevas clases basadas en clases existentes, heredando atributos y métodos. El polimorfismo permite que objetos de diferentes clases se comporten de manera similar a través de una interfaz común.


Python

    
        class Estudiante(Persona):
    def __init__(self, nombre, edad, matricula):
        super().__init__(nombre, edad)
        self.matricula = matricula

    def obtener_info(self):
        return f"Nombre: {self.nombre}, Edad: {self.obtener_edad()}, Matrícula: {self.matricula}"
