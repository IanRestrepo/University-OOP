# ¿Que es la Abstracción?


es el proceso de seleccionar las características y funciones más relevantes de una entidad real para modelarla como una clase. Por ejemplo, si queremos representar un automóvil, podemos abstraer sus atributos como marca, modelo, color, velocidad, etc., y sus métodos como arrancar, frenar, acelerar, etc. La abstracción nos permite simplificar la complejidad del problema y enfocarnos en lo esencial. 

    
        class Vehiculo {
            constructor(marca, modelo) {
              this.marca = marca;
              this.modelo = modelo;
              this.encendido = false;
            }
          
            encender() {
              if (!this.encendido) {
                this.encendido = true;
                console.log(`El ${this.marca} ${this.modelo} ha sido encendido.`);
              } else {
                console.log(`El ${this.marca} ${this.modelo} ya está encendido.`);
              }
            }
          
            apagar() {
              if (this.encendido) {
                this.encendido = false;
                console.log(`El ${this.marca} ${this.modelo} ha sido apagado.`);
              } else {
                console.log(`El ${this.marca} ${this.modelo} ya está apagado.`);
              }
            }
          }
          
          // Crear objetos de la clase Vehiculo
          const miAuto = new Vehiculo("Toyota", "Corolla");
          const miMoto = new Vehiculo("Honda", "CBR");
          
          // Interactuar con los objetos
          miAuto.encender();
          miMoto.encender();
          
          miAuto.apagar();
          miMoto.apagar();

## Su Sintaxis


la encapsulación es uno de los conceptos fundamentales en la programación orientada a objetos (POO) que se refiere a la idea de ocultar los detalles internos de una clase y proporcionar una interfaz controlada para interactuar con ella. Esto se logra a través del uso de modificadores de acceso y métodos para limitar el acceso directo a los atributos y comportamientos internos de un objeto. En muchos lenguajes de programación, como Java, C++ y Python, se implementa mediante la definición de clases y el uso de sus miembros públicos, privados y protegidos.


JavaScript

    
        class MiClase {
            constructor() {
                // Atributo encapsulado
                this._atributoEncapsulado = 0;
            }
        
            // Método para obtener el atributo encapsulado
            obtenerAtributo() {
                return this._atributoEncapsulado;
            }
        
            // Método para establecer el atributo encapsulado
            establecerAtributo(nuevoValor) {
                this._atributoEncapsulado = nuevoValor;
            }
        }
        
        // Crear una instancia de la clase
        const instancia = new MiClase();
        
        // Acceder a los métodos públicos para interactuar con el atributo encapsulado
        console.log(instancia.obtenerAtributo()); // Imprime 0
        instancia.establecerAtributo(42);
        console.log(instancia.obtenerAtributo()); // Imprime 42

Java

    
        public class MiClase {
            // Atributos privados
            private int atributoPrivado;
        
            // Constructor
            public MiClase(int valor) {
                this.atributoPrivado = valor;
            }
        
            // Método público para acceder al atributo privado
            public int getAtributoPrivado() {
                return atributoPrivado;
            }
        
            // Método público para modificar el atributo privado
            public void setAtributoPrivado(int nuevoValor) {
                this.atributoPrivado = nuevoValor;
            }
        }

Python

    
    class MiClase:
        def __init__(self):
            # Atributo encapsulado
            self._atributo_encapsulado = 0

        # Método para obtener el atributo encapsulado
        def obtener_atributo(self):
            return self._atributo_encapsulado

        # Método para establecer el atributo encapsulado
        def establecer_atributo(self, nuevo_valor):
            self._atributo_encapsulado = nuevo_valor

        # Crear una instancia de la clase
        instancia = MiClase()

        # Acceder a los métodos públicos para interactuar con el atributo encapsulado
        print(instancia.obtener_atributo())  # Imprime 0
        instancia.establecer_atributo(42)
        print(instancia.obtener_atributo())  # Imprime 42
