Programación Orientada A Objetos📦

La programación orientada a objetos (POO) es una forma de escribir código en la que agrupamos información y las cosas que podemos hacer con esa información en pequeñas piezas llamadas "objetos".

Imagina que estás haciendo un juego de video. En lugar de tener una lista enorme de instrucciones confusas, puedes crear objetos para las diferentes partes del juego. Por ejemplo, podrías tener un objeto llamado "jugador" que tenga información sobre el personaje principal, como su nombre, nivel y habilidades. También podrías tener objetos para los enemigos, los elementos del juego y más.
