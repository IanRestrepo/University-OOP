Programación Orientada a Objetos: Herencia

# ¿Que es la Herencia?


Esto permite organizar las clases en una jerarquía de generalización-especialización, donde las clases más generales se llaman superclases o clases base, y las más específicas se llaman subclases o clases derivadas. Por ejemplo, si tenemos una clase vehículo como superclase, podemos crear subclases como automóvil, motocicleta, bicicleta, etc., que heredan sus atributos y métodos comunes, y agregan otros propios. 

    
        class Vehiculo {
            constructor(placas) {
              this.placas = placas;
              this.ingresado = false;
            }
          
            ingresar() {
              if (!this.ingresado) {
                this.ingresado = true;
                console.log(`El vehículo con placas ${this.placas} ingresó al parqueadero.`);
              } else {
                console.log(`El vehículo con placas ${this.placas} ya está en el parqueadero.`);
              }
            }
          
            salir() {
              if (this.ingresado) {
                this.ingresado = false;
                console.log(`El vehículo con placas ${this.placas} salió del parqueadero.`);
              } else {
                console.log(`El vehículo con placas ${this.placas} no está en el parqueadero.`);
              }
            }
          }
          
          class Automovil extends Vehiculo {}
          
          const auto1 = new Automovil("ABC123");
          const auto2 = new Automovil("XYZ789");
          
          auto1.ingresar();
          auto2.ingresar();
          auto1.salir();
          auto2.salir();
