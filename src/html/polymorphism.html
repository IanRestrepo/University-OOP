# ¿Que es el polimorfismo?


Es la capacidad de que un mismo objeto pueda comportarse de diferentes formas según el contexto en el que se encuentre. Esto se logra mediante la sobrecarga y la sobreescritura de métodos. 

    
        class Animal {
    hablar() {
        console.log("Sonido desconocido");
    }
}

class Perro extends Animal {
    hablar() {
        console.log("Woof!");
    }
}

class Gato extends Animal {
    hablar() {
        console.log("Meow!");
    }
}

class Vaca extends Animal {
    hablar() {
        console.log("Moo!");
    }
}

function hacerHablar(animal) {
    animal.hablar();
}

// Crear instancias de las clases derivadas
const perro = new Perro();
const gato = new Gato();
const vaca = new Vaca();

// Llamar al método haciendo uso del polimorfismo
hacerHablar(perro); // Imprimirá "Woof!"
hacerHablar(gato);  // Imprimirá "Meow!"
hacerHablar(vaca);  // Imprimirá "Moo!"

## Su Sintaxis


La sintaxis del polimorfismo puede variar según el lenguaje de programación que estés utilizando, pero el concepto general es que se trata de la capacidad de objetos de diferentes clases de ser tratados como si fueran objetos de la misma clase base. 

    
        class Forma {
            calcularArea() {
                return 0.0;
            }
        }
        
        class Circulo extends Forma {
            constructor(radio) {
                super();
                this.radio = radio;
            }
        
            calcularArea() {
                return Math.PI * this.radio * this.radio;
            }
        }
        
        class Rectangulo extends Forma {
            constructor(ancho, alto) {
                super();
                this.ancho = ancho;
                this.alto = alto;
            }
        
            calcularArea() {
                return this.ancho * this.alto;
            }
        }
        
        // Crear instancias de las clases derivadas
        const forma1 = new Circulo(5.0);
        const forma2 = new Rectangulo(4.0, 6.0);
        
        // Llamar al método calcularArea() haciendo uso del polimorfismo
        console.log("Área del círculo: " + forma1.calcularArea());
        console.log("Área del rectángulo: " + forma2.calcularArea());
